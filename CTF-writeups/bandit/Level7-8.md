# Bandit Level 7 → Level 8

## Objective
Extract a specific line from a large text file using pattern matching.

## Concepts Learned
- Searching inside files with grep
- Using literal strings as search patterns
- Handling large files efficiently

## Tools & Commands Introduced
- grep – print lines matching a pattern
- Basic string matching with grep "pattern" file

## Approach
1. Identify the target file (data.txt).
2. Use grep with the exact keyword to filter the relevant line.
3. Extract the password from the matched line.

## Example Command
grep "search_term" filename

## Key Takeaway
- grep is a fundamental tool for text processing and log analysis.
- Exact string matching is often the first step in data extraction.
